# Helper Functions & UI Components

This directory contains shared utility functions and UI components used across the FinTrack Pro application.

- `ui.py`: Provides reusable `Rich` components for enhanced terminal output, such as formatted tables, panels, and loading spinners.
- `helpers.py`: Contains general-purpose helper functions, such as date formatting, currency formatting, and other common utilities.

These modules ensure consistency and reusability of common functionalities and user interface elements.
